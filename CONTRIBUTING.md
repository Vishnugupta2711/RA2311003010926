# Contributing

Thank you for your interest in contributing to Nexus Backend Core. 

Because this is an evaluation track project, PRs are currently closed. However, if you are cloning this for your own development:

1. **Fork the repo.**
2. **Create a new branch** (`git checkout -b feat/your-feature`).
3. **Commit your changes** following the Conventional Commits specification.
4. **Push to the branch** (`git push origin feat/your-feature`).
5. **Open a Pull Request.**

Please ensure all tests pass and that you adhere to the zero-external-dependency rule unless strictly necessary.
